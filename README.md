# E-commerce Sales Analysis Dashboard

## Project Overview
This project analyzes e-commerce sales data to identify business insights such as top-selling products, monthly sales trends, customer segments, and profit performance.

## Tools & Technologies Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Plotly
- Jupyter Notebook

## Key Features
- Sales trend analysis by month
- Top products by revenue
- Category-wise performance
- Profit and loss insights
- Interactive visualizations using Plotly

## Project Files
- E-Commerce data analysis.ipynb
- Superstore.csv
- README.md

## Business Insights
- Identified high-performing categories
- Found peak sales months
- Analyzed profit margins
- Suggested data-driven improvements
